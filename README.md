# 24V 250W Brushed DC Motor Controller

This project focuses on the design and development of a complete control system for a 24V, 250W brushed DC motor. The system includes power electronics, driver electronics, and a microcontroller-based control board.

## Project Overview

The goal of this project is to design a functional and safe DC motor control system including:

- Control Board (Microcontroller + Buttons)
- Driver Electronics (MOSFET Gate Driver)
- Power Electronics (MOSFET Power Stage)
- DC Motor (24V, 250W Brushed Motor)
- Power Supply

The system is designed as an open-frame laboratory prototype mounted on a wooden board.

## Features

- 24V Brushed DC Motor Control
- Speed Control (PWM)
- Push Button Control (Start / Stop / Speed Up / Speed Down)
- Overcurrent Protection
- Safe Hardware Design
- Modular Block Architecture
